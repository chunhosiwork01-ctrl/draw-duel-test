# Hong Kong Housing Reality Quiz

A single-file interactive quiz about Hong Kong housing affordability.

## Files

- `index.html`: deployment entry page
- `hk_survival.html`: main quiz

## Deploy To GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository root:
   - `index.html`
   - `hk_survival.html`
   - `README.md`
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Source`, choose `Deploy from a branch`.
5. Select:
   - Branch: `main`
   - Folder: `/root`
6. Save and wait a few minutes.
7. Your public URL will look like:
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/`
